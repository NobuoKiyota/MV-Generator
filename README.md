# 🎬 自走型MVジェネレーター (MV-Generator)

音源ファイル（mp3/wav等）と歌詞を入力するだけで、AI（Gemini & Stable Diffusion / Imagen）が曲の展開・テンポ（BPM）を自動解析し、音楽に完全同期したVコンテ（仮MV）用スライドショー動画を一発で自動生成するツールです。

サウンドデザイナー向けの設計となっており、テンポ（BPM）や拍（ビート）に正確に吸い付くカット割りが特徴です。

---

## 🌟 主な特徴

1.  **ハイブリッド画像生成 (ローカルGPU優先 / クラウド Imagen)**
    *   PCにNVIDIA製GPU（RTX 3060/4060など）が搭載されている場合、自動的にローカルの **Stable Diffusion (DreamShaper-8)** を呼び出し、**完全無料・無制限・爆速（1枚あたり2〜3秒）**で高品質な画像を生成します。
    *   GPUが無い環境や、APIでの生成を行いたい場合は、自動的に **Google AI Studio (Imagen 4.0)** のクラウドAPIへフォールバックします。
2.  **対話的な画像プロンプト改善**
    *   生成された絵を見ながら、日本語で「もっと雨を激しく」「夜のネオン街にして」と指示を入れるだけで、Geminiが自動的にImagen/SD用の高精細な英語プロンプトへとブラッシュアップしてくれます。
3.  **仮画像（プレースホルダー）対応のスライドショー動画合成**
    *   画像を生成していないカットには、自動で歌詞や構成情報が書かれた黒背景の仮画像を差し込みます。「サビの重要な数枚だけ画像を生成し、その他は仮画像のまま、曲のタイミングに合わせて再生される仮MV（Vコンテ）」を数十秒でレンダリングできます（MoviePyによる爆速マルチスレッド処理）。
4.  **インスト曲への完全対応**
    *   歌詞がないインストゥルメンタル曲でも、音源のビート（拍）とテンポを自動解析し、映像プロットの設定に沿って、曲の展開に同期した美しいスライドショーを自動設計します。
5.  **ZIPパッケージ一括保存・読込**
    *   JSONタイムラインデータ、BGM音源（mp3/wav等）、および生成済みのカット画像をすべてまとめて1つの `.zip` パッケージファイルとして保存・読み込み可能です。別のPCへの移行もZIPひとつで完結します。

## 🛠️ セットアップと起動方法

1.  **リポジトリのクローン**
    ```bash
    git clone https://github.com/NobuoKiyota/MV-Generator.git
    cd MV-Generator
    ```

2.  **APIキーの設定**
    プロジェクトルートに `.env` ファイルを作成し、Google AI StudioのGemini APIキーを設定します（※画面上で空欄にした場合はこのキーが自動で使用されます）。
    ```env
    GEMINI_API_KEY=あなたのAPIキー(AIzaSy...)
    ```

3.  **ワンクリック起動**
    Windows環境であれば、プロジェクトルートの **`run.bat`** をダブルクリックするだけで、自動的にブラウザが立ち上がりStreamlitアプリが起動します。

---

## 📁 主要ファイル構成

*   `app.py` : StreamlitによるGUIアプリケーション。
*   `utils.py` : 音響解析、構造化歌詞パース、Gemini連携（タイムライン設計・プロンプト改善）、ZIPパッケージ管理。
*   `video_utils.py` : ローカル画像生成（Stable Diffusion）、仮画像生成（Pillow）、動画合成（MoviePy）。
*   `requirements.txt` : 依存パッケージ一覧。
*   `run.bat` : アプリ起動バッチスクリプト。
